# Netflix 雙語字幕 Chrome 擴充功能

在 Netflix 播放頁面上同時顯示兩種語言的字幕，適合語言學習與多語需求。

## 功能

- 在 Netflix 原生字幕上方（或下方）疊加第二語言字幕
- 支援所有 Netflix 提供的字幕語言
- 可調整字體大小、透明度與顯示位置
- 支援 WebVTT 與 TTML/DFXP 字幕格式

## 安裝

此擴充功能尚未上架 Chrome 線上應用程式商店，需手動載入：

1. 下載或 clone 此 repository
2. 開啟 Chrome，前往 `chrome://extensions/`
3. 右上角啟用「開發人員模式」
4. 點擊「載入未封裝項目」，選擇此資料夾

## 使用方式

1. 前往 Netflix 並開始播放任何影片
2. **先在 Netflix 播放器內開啟原生字幕**（任意語言皆可）——這樣擴充功能才能攔截到字幕檔案
3. 點擊瀏覽器右上角的擴充功能圖示
4. 從下拉選單選擇第二字幕語言
5. 字幕會自動疊加在畫面上

> **提示**：如果選擇語言後字幕未立即出現，稍等幾秒或在 Netflix 播放器切換一次該語言的原生字幕，擴充功能即可攔截並顯示。

## 設定選項

| 選項 | 說明 |
|------|------|
| 開關 | 暫時隱藏/顯示第二字幕，不影響已載入的字幕資料 |
| 字體大小 | 小 / 中 / 大 |
| 字幕位置 | 原生字幕上方 / 畫面底部 |
| 透明度 | 10%–100% |

## 運作原理

擴充功能透過監聽 Netflix 播放器的網路請求來取得字幕內容。當 Netflix 下載字幕檔案時，擴充功能會攔截並快取該語言的內容，再以自訂樣式疊加在播放器畫面上。擴充功能**不會上傳任何資料**，所有處理均在本機完成。

## 已知限制

- 需要 Netflix 的原生字幕功能正常運作（用於觸發字幕下載）
- Netflix 頁面重新整理後需重新選擇第二字幕語言
- 部分地區或裝置版本的 Netflix 播放器結構不同，可能影響字幕偵測
